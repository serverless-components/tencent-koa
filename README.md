# 腾讯云 Koa 组件

简体中文 | [English](https://github.com/serverless-components/tencent-koa/tree/master/README.en.md)

## 简介

使用腾讯云 Koa 组件，可快速的在腾讯云创建，配置和管理一个 [Koa 框架](https://koajs.com/) 服务。

## 快速开始

```bash
# 安装 serverless
npm install -g serverless

# 下载 koa-demo
serverless init koa-demo

# 部署
cd koa-demo
serverless deploy
```
